# 数据库说明

## 1. 当前数据库配置

数据库配置从以下文件读取：

```text
backend/src/config/config.local.ts
```

当前本地数据库配置为：

- 数据库类型：MySQL
- 主机：`127.0.0.1`
- 端口：`3306`
- 数据库名：`cool`
- 账号：`root`
- 密码：`123456`
- 字符集：`utf8mb4`

后端公共端口配置在：

```text
backend/src/config/config.default.ts
```

当前后端端口：

```text
3000
```

## 2. 当前数据库数据情况

当前数据库里已有分类和商品数据。

已通过浏览器验证：

```text
http://127.0.0.1:3000/api/app/wudong/product-categories/list
```

该接口能返回：

- `code=1000`
- `data` 中有分类数据

```text
http://127.0.0.1:3000/api/app/wudong/products/page?page=1&pageSize=5
```

该接口能返回：

- `code=1000`
- `data.total=8`
- `data.list` 中有商品数据

## 3. 商品相关表

根据 `backend/src/modules/wudong/entity/` 下实体文件确认，当前商品相关表包括：

- `product_category`：商品分类表
- `product`：商品表
- `product_sku`：商品 SKU / 库存 / 价格相关表
- `product_image`：商品图片表
- `favorite`：收藏表

## 4. 初始化 SQL 导出情况

已成功使用本机 `mysqldump` 导出当前项目相关数据表。

导出文件：

```text
D:\临时文件\水课任务\人工智能实践\init-data.sql
```

使用的导出工具：

```text
D:\mysql\MySQL Server 8.0\bin\mysqldump.exe
```

导出的表：

- `product_category`
- `product`
- `product_sku`
- `product_image`
- `favorite`

导出命令参考：

```powershell
& 'D:\mysql\MySQL Server 8.0\bin\mysqldump.exe' --host=127.0.0.1 --port=3306 --user=root --password=123456 --default-character-set=utf8mb4 --single-transaction --skip-lock-tables --column-statistics=0 cool product_category product product_sku product_image favorite --result-file='D:\临时文件\水课任务\人工智能实践\init-data.sql'
```

说明：命令行中直接写密码会出现 MySQL 安全提示，这是正常警告，不影响导出结果。

## 5. 导入 SQL 参考

如果接手同学需要恢复当前商品相关数据，可在已有 MySQL 环境中执行类似命令：

```bat
mysql -h 127.0.0.1 -P 3306 -u root -p cool < init-data.sql
```

导入前请确认目标库和目标表，避免覆盖自己已有数据。

```text

```

```bat
curl "http://127.0.0.1:3000/api/app/wudong/product-categories/list"
```
