# 06 微信小程序端页面与交互设计

本文件适用于第 1 组“衣·非遗商品模块”。

## 1. 技术方案

* 默认方案：原生微信小程序
* 请求方式：`wx.request`
* 本地调试后端地址：`http://localhost:3000`
* 不改为 UniApp 或 Taro

## 2. 页面清单

| 页面 | 建议路径 | 说明 |
| --- | --- | --- |
| 首页/商品列表页 | `/pages/products/index` | 展示分类和商品列表 |
| 商品详情页 | `/pages/products/detail` | 展示详情、SKU、收藏按钮 |
| 我的收藏页 | `/pages/favorites/index` | 展示当前用户收藏的商品 |

## 3. 页面设计

## 3.1 首页/商品列表页

### 页面路径

`/pages/products/index`

### 页面功能

* 展示分类列表
* 按分类筛选商品
* 支持分页加载商品
* 点击进入商品详情页

### 页面数据

* 分类列表
* 商品列表
* 当前分类 ID
* 当前页码与是否还有更多

### 调用接口

* `GET /api/product-categories`
* `GET /api/products`

### 用户操作流程

1. 页面加载时先请求分类与第一页商品
2. 用户点击分类后刷新列表
3. 用户下拉或触底时加载下一页
4. 用户点击商品进入详情页

### loading

* 首屏页面 loading
* 上拉加载更多 loading

### 空状态

* 当前分类暂无商品

### 错误状态

* 接口失败时显示重试按钮

### 未登录处理

* 本页无需登录

## 3.2 商品详情页

### 页面路径

`/pages/products/detail?id={productId}`

### 页面功能

* 展示商品主图和图片列表
* 展示商品名称、工艺介绍、传承人信息
* 展示 SKU、价格、库存、售罄状态
* 支持收藏和取消收藏

### 页面数据

* 商品详情对象
* SKU 列表
* 图片列表
* 当前选中 SKU
* 收藏状态

### 调用接口

* `GET /api/products/{productId}`
* `GET /api/products/{productId}/skus`
* `GET /api/products/{productId}/images`
* `GET /api/favorites/{productId}/status`
* `POST /api/favorites`
* `DELETE /api/favorites/{productId}`

### 用户操作流程

1. 页面进入后加载商品详情、SKU 和图片
2. 用户选择某个 SKU，页面刷新价格、库存和售罄文案
3. 若用户已登录，则同步查询收藏状态
4. 点击收藏按钮时发起收藏或取消收藏请求

### loading

* 首屏 loading
* 收藏按钮 loading

### 空状态

* 商品不存在或已下架

### 错误状态

* 接口失败时提示“加载失败，请重试”

### 未登录处理

* 未登录用户可以浏览详情
* 点击收藏时提示“请先登录后再收藏”
* 不应伪造本地收藏成功状态

## 3.3 我的收藏页

### 页面路径

`/pages/favorites/index`

### 页面功能

* 展示当前登录用户收藏的商品
* 支持取消收藏
* 点击进入商品详情页

### 页面数据

* 收藏商品列表
* 当前页码
* 是否还有更多

### 调用接口

* `GET /api/favorites`
* `DELETE /api/favorites/{productId}`

### 用户操作流程

1. 页面进入时校验是否有用户登录态
2. 有登录态时拉取收藏列表
3. 点击商品跳转详情页
4. 点击取消收藏后刷新当前列表

### loading

* 首屏 loading
* 下拉分页 loading
* 单项取消收藏 loading

### 空状态

* 暂无收藏商品

### 错误状态

* 接口失败时提示并允许重试

### 未登录处理

* 未登录时展示引导文案，不直接请求收藏接口

## 4. 商品详情页关键展示项

商品详情页必须支持以下信息：

* 商品主图
* 商品图片
* 商品名称
* 工艺介绍
* 传承人信息
* SKU 选择
* 价格
* 库存
* 售罄状态
* 收藏按钮

## 5. 本地调试说明

在微信开发者工具本地调试时：

* 使用 `wx.request`
* 后端地址为 `http://localhost:3000`
* 需要开启“不校验合法域名”
* 真机测试不能直接使用电脑 `localhost`
* 真机联调时需替换为局域网地址、穿透地址或部署地址

## 6. 最小可验收策略

* 小程序只覆盖商品浏览与收藏，不扩展购物车和下单
* 登录流程属于系统支撑能力，当前文档只约束登录后行为
