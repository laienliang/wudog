# 乌东系统 - 图片素材目录

> 所有图片上传后请将路径映射到此目录

## 目录结构

```
docs/assets/
├── README.md              ← 本文件
├── clothing/              ← 衣（非遗商品）
│   ├── goods/             ← 商品主图
│   ├── sku/               ← SKU 规格图
│   └── category/          ← 分类图标
├── food/                  ← 食（餐饮美食）
│   ├── restaurant/        ← 餐厅主图
│   ├── dish/              ← 菜品图
│   └── agriculture/       ← 农产品图
├── hostel/                ← 住（民宿住宿）
│   ├── main/              ← 民宿外观图
│   └── room/              ← 房型图
├── scenic/                ← 行（旅行出行）
│   ├── main/              ← 景点主图
│   └── detail/            ← 景点细节图
└── article/               ← 社区文章配图
```

## 图片命名规范

```
{模块}_{实体}_{序号}.{格式}
例：clothing_goods_001.jpg
    food_dish_01_suantangyu.jpg
    hostel_main_01.jpg
```

## 图片规格要求

| 用途 | 尺寸 | 格式 | 大小 |
|------|------|------|------|
| 商品/餐厅/民宿主图 | 800x800 | JPG | <500KB |
| SKU 图 | 400x400 | JPG | <300KB |
| 菜品图 | 600x400 | JPG | <400KB |
| 房型图 | 800x500 | JPG | <500KB |
| 景点图 | 1200x600 | JPG | <800KB |
| 文章配图 | 1200x630 | JPG | <500KB |
| 分类图标 | 64x64 | PNG | <50KB |
