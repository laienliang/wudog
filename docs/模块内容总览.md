# 乌东系统 - 模块内容总览

> 基于 IMA 知识库「OPC软3-乌东」中的系统结构整理

## 一、衣（非遗商品）

### 数据表结构

| 表名 | 说明 | 关键字段 |
|------|------|----------|
| `clothing_category` | 商品分类 | parentId, name, icon, sort, status |
| `clothing_goods` | 非遗商品 | categoryId, merchantId, title, subtitle, price, marketPrice, stock, sales, craftIntro, inheritorName, status |
| `clothing_goods_sku` | 商品SKU规格 | goodsId, specName, price, stock, image |
| `clothing_review` | 商品评价 | goodsId, userId, rating, content, images |
| `clothing_collect` | 商品收藏 | userId, goodsId |

### 核心功能
- 非遗商品展示与分类管理
- 商品规格（SKU）管理
- 传承人信息展示
- 手工艺介绍
- 用户评价与收藏

---

## 二、食（餐饮美食）

### 数据表结构

| 表名 | 说明 | 关键字段 |
|------|------|----------|
| `food_agriculture_category` | 农产品分类 | parentId, name, icon, sort |
| `food_restaurant` | 餐厅 | merchantId, name, description, mainImage, address, businessHours, capacity, rating, status |
| `food_dish` | 菜品 | restaurantId, name, price, isSignature, sort |
| `food_agriculture_goods` | 农产品 | categoryId, merchantId, name, price, mainImage, specification, origin, shelfLife, stock, status |
| `food_time_slot` | 用餐时段 | restaurantId, name, maxBookings |
| `food_review` | 餐厅/菜品评价 | restaurantId, goodsId, userId, rating, content, images |
| `food_collect` | 美食收藏 | userId, targetId |

### 核心功能
- 餐厅管理与展示
- 菜品管理（支持招牌菜标识）
- 农产品商城
- 用餐预约（时段管理）
- 用户评价与收藏

---

## 三、住（民宿住宿）

### 数据表结构

| 表名 | 说明 | 关键字段 |
|------|------|----------|
| `lodging_hostel` | 民宿 | merchantId, name, address, mainImage, images[], description, rating, styleTags, facilityTags, status |
| `lodging_room_type` | 房型 | hostelId, name, bedType, area, capacity, price, facilities, mainImage, stock |
| `lodging_calendar` | 房态日历 | roomTypeId, date, availableStock, price, isAvailable |
| `lodging_hostel_policy` | 民宿政策 | hostelId, checkInTime, checkOutTime, petPolicy, includeBreakfast, deposit |
| `lodging_review` | 民宿评价 | orderId, hostelId, userId, rating, content, images |
| `lodging_collect` | 民宿收藏 | userId, hostelId |

### 核心功能
- 民宿信息管理（风格标签、设施标签）
- 房型管理与价格设置
- 每日房态与动态定价
- 入住/退房政策配置
- 用户评价与收藏

---

## 四、行（旅行出行）

### 数据表结构

| 表名 | 说明 | 关键字段 |
|------|------|----------|
| `travel_scenic` | 景点 | merchantId, name, address, mainImage, description, openHours, rating, status |
| `travel_ticket_type` | 票务类型 | scenicId, name, price, stock, validityRule |
| `travel_route` | 旅行路线 | merchantId, title, mainImage, days, price, includes, notes, detailContent, status |
| `travel_route_day` | 每日行程 | routeId, day, description, spots, meals, accommodation |
| `travel_e_ticket` | 电子票 | orderNo, targetId, qrCode, status, useDate, verifyTime |
| `travel_guide` | 旅行攻略 | title, departure, transport, duration, cost, content, images[] |
| `travel_review` | 旅行评价 | targetId, userId, rating, content |
| `travel_collect` | 旅行收藏 | userId, targetType(1=景点,2=路线), targetId |

### 核心功能
- 景点管理与展示
- 电子票务系统（二维码核销）
- 旅行路线规划（按天展示行程）
- 旅行攻略分享
- 用户评价与收藏

---

## 五、社区（UGC内容）

### 数据表结构

| 表名 | 说明 | 关键字段 |
|------|------|----------|
| `community_topic` | 话题标签 | name, description, icon, followers, articles, isRecommended |
| `community_tag` | 内容标签 | name |
| `community_article` | 游记/文章 | userId, title, content, images[], videoUrl, topicIds[], relatedPlaceType, relatedPlaceId, likes, comments, collects, views, status |
| `community_comment` | 评论 | articleId, userId, content |
| `community_like` | 点赞 | targetId, targetType, userId |
| `community_follow` | 关注 | userId, targetUserId, topicId |
| `community_collect` | 收藏 | userId, targetId, targetType |
| `community_report` | 举报 | targetId, targetType, reason, userId |
| `community_image` | 图片 | userId, imageUrl, tags |
| `community_video` | 视频 | userId, videoUrl, cover, duration |

### 核心功能
- 游记文章发布（支持图文+视频）
- 话题系统（关注+推荐）
- 内容标签体系
- 互动功能（点赞、评论、收藏、关注）
- 内容审核（举报机制）
- 关联本地商家（餐厅/民宿/景点）
