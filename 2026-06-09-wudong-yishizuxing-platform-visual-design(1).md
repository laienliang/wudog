## 1. 设计理念与品牌调性

### 1.1 设计理念

> **"苗寨有银，文化有纹"**
> 
> 以贵州苗族文化为灵感来源，将银饰的冷冽光泽、蜡染的靛蓝、刺绣的暖色调融入现代数字界面，营造"民族文化 × 数字文旅"的独特视觉体验。

### 1.2 品牌关键词

| 关键词 | 视觉表现 |
|--------|---------|
| 民族文化 | 银饰、蜡染、刺绣纹样、苗银冷色调 |
| 山野自然 | 梯田绿、远山青、暖阳橙 |
| 数字现代 | 干净留白、扁平化、几何线条 |
| 信任温暖 | 暖色调辅助色、亲切文案、圆润元素 |

### 1.3 调性原则

- **主调**：暖中性 + 苗银冷色点缀
- **气质**：年轻、有文化感、值得信赖、易亲近
- **风格**：扁平化为主，关键品牌元素可使用微立体/纹理
- **情感**：唤起用户对苗寨自然与人文的向往

## 2. 色彩系统

### 2.1 品牌主色

| 颜色 | 色值 | 名称 | 使用场景 |
|------|------|------|---------|
| **苗银主色** | `#1F5FA8` | 苗银蓝 | 主按钮、链接、关键操作、品牌 Logo |
| **苗银深色** | `#0E3D75` | 苗银深蓝 | 按钮按下态、强调文字、标题 |
| **苗银浅色** | `#E8F1FB` | 苗银浅蓝 | 背景填充、选中态背景、Tag 背景 |

> **设计理由**：苗银蓝取自苗族银饰冷冽光泽与蜡染靛蓝的结合，既体现民族文化，又具有数字感，适合作为主品牌色。

| 颜色 | 色值 | 名称 | 使用场景 |
|------|------|------|---------|

| **梯田绿** | `#6B8E3D` | 梯田翠绿 | 成功状态、确认按钮、自然类目 |

| **古木棕** | `#7A5230` | 苗寨木屋棕 | 文案点缀、二级标题 |

### 2.3 中性色

| 颜色 | 色值 | 使用场景 |
|------|------|---------|
| 主文字 | `#1A1A1A` | 标题、正文 |
| 次要文字 | `#595959` | 副标题、辅助信息 |

| 弱化文字 | `#BFBFBF` | 禁用态 |

| 三级背景 | `#FAFAFA` | 内容区背景 |

### 2.4 功能色

| 颜色 | 色值 | 使用场景 |
|------|------|---------|
| 成功 | `#52C41A` | 操作成功提示 |
| 警告 | `#FAAD14` | 警告信息 |
| 错误 | `#FF4D4F` | 错误提示、删除操作 |
| 信息 | `#1F5FA8` | 提示信息（与主色一致） |
| 链接 | `#1F5FA8` | 文本链接（与主色一致） |

### 2.5 色彩使用规范

1. **同一页面主色不超过 2 种**，避免色彩杂乱
2. **主色使用占比 5-15%**，辅色 5-10%，中性色占主导
3. **色值透明度规范**：
   - 悬浮遮罩：`rgba(0, 0, 0, 0.5)`
   - 弹窗遮罩：`rgba(0, 0, 0, 0.6)`
   - 主色透明度变体：`#1F5FA8` 配 `20% / 40% / 60% / 80%` 透明度

### 2.6 深色模式（可选，V2 考虑）

| 颜色 | 色值 |
|------|------|
| 主背景 | `#121212` |
| 卡片背景 | `#1E1E1E` |
| 主文字 | `#F5F5F5` |
| 次要文字 | `#BFBFBF` |
| 主色 | `#3B7BC5`（比浅色模式亮 20%） |

## 4. 栅格与间距

### 4.1 8px 基准栅格

所有间距、尺寸均基于 **8px 基准栅格**：

| 名称 | 数值 | 用途 |
|------|------|------|
| 2xs | 4px | 图标与文字间距、紧凑排列 |
| xs | 8px | 列表项内边距、标签内边距 |
| sm | 12px | 卡片内组件间距 |
| md | 16px | 卡片内边距、模块内组件间距 |
| lg | 24px | 区块间距 |
| xl | 32px | 大区块间距 |
| 2xl | 48px | 页面级间距 |
| 3xl | 64px | 章节间距 |

### 4.2 页面边距

| 端 | 页面左右边距 |
|----|------------|
| 小程序 | 16px |
| PC 端 | 24px（容器居中，最大宽度 1200px） |
| 管理后台 | 24px（侧边栏 220px + 主内容区） |

### 4.3 栅格系统

- **小程序端**：4 列栅格（列宽 = (屏幕宽 - 32) / 4），列间距 8px
- **PC 端**：12 列栅格，列宽弹性，最大容器 1200px
- **管理后台**：固定侧边栏 220px + 主内容区自适应

## 6. 图标系统

### 6.1 图标风格

- **线性图标为主**，2px 描边
- 端点圆角处理
- 关键图标（Logo、加载）可使用填充式
- 图标尺寸：16px / 20px / 24px / 32px / 48px

### 6.2 推荐图标库

| 端 | 图标库 |
|----|-------|
| 小程序 | Iconfont / IconPark / 自研 SVG |
| PC 端 | IconPark / Element Plus Icons / 自研 SVG |
| 管理后台 | Element Plus Icons / Ant Design Icons |

### 6.3 自研图标规范

- SVG 格式
- 24×24px 视图框
- 2px 描边（圆角端点）
- 单一颜色，currentColor
- 提供 outline（线性）和 filled（填充）两套

## 7. 按钮组件

### 7.1 按钮类型

| 类型 | 样式 | 用途 |
|------|------|------|
| 主按钮 | 实心苗银蓝 | 主要操作（立即购买、提交、确认） |

| 文字按钮 | 无背景无边框 | 辅助操作（查看更多、了解详情） |
| 危险按钮 | 实心错误色 | 删除、清空 |
| 禁用按钮 | 灰色背景 | 不可点击 |

### 7.2 按钮尺寸

| 尺寸 | 高度 | 内边距 | 字号 | 圆角 |
|------|------|--------|------|------|
| 小 | 28px | 12px | 12px | 4px |
| 默认 | 36px | 16px | 14px | 6px |
| 中 | 40px | 20px | 14px | 8px |
| 大 | 48px | 24px | 16px | 8px |
| 块状 | 48px | 0 | 16px | 8px（占满父容器） |

### 7.3 按钮状态

| 状态 | 样式变化 |
|------|---------|
| 默认 | 实心 `#1F5FA8`，文字 `#FFFFFF` |
| 悬浮 | 背景 `#3B7BC5`（亮度 +10%） |
| 按下 | 背景 `#0E3D75`（亮度 -10%） |
| 禁用 | 背景 `#BFBFBF`，文字 `#FFFFFF`，不可点击 |
| 加载 | 右侧出现 Loading 旋转图标，文字变为"处理中..." |

### 7.4 按钮文案规范

- 使用动词："立即购买"、"加入购物车"、"提交订单"
- 避免模糊用词："确定"、"好的"
- 主按钮文案 ≤ 6 个汉字

## 14. 图片与多媒体

### 14.1 图片规格

| 类型 | 尺寸 | 格式 | 大小限制 |
|------|------|------|---------|
| 商品主图 | 800×800px | jpg/png/webp | ≤ 200KB |
| 商品列表缩略图 | 400×400px | jpg/png/webp | ≤ 100KB |
| Banner 图 | 750×320px（小程序）/ 1920×600px（PC） | jpg | ≤ 300KB |
| 头像 | 200×200px | jpg/png | ≤ 50KB |
| 游记图片 | 1080×1080px | jpg | ≤ 500KB |
| 视频封面 | 720×720px | jpg | ≤ 100KB |

### 14.2 图片加载

- 使用懒加载
- 默认占位：`#F0F0F0` 纯色 + 加载图标
- WebP 优先，降级到 jpg

## 15. 动效规范

### 15.1 动画时长

| 类型 | 时长 | 用途 |
|------|------|------|
| 即时 | 100ms | 按钮按下反馈 |
| 快速 | 200ms | 状态切换、颜色变化 |
| 标准 | 300ms | 弹窗、Drawer、Tab 切换 |
| 慢速 | 500ms | 页面切换、骨架屏 |

### 15.2 缓动函数

| 名称 | 函数 | 用途 |
|------|------|------|
| 标准 | `cubic-bezier(0.4, 0, 0.2, 1)` | 默认缓动 |
| 进入 | `cubic-bezier(0, 0, 0.2, 1)` | 元素进入 |
| 离开 | `cubic-bezier(0.4, 0, 1, 1)` | 元素离开 |

### 15.3 转场动效

- 页面切换：淡入淡出 200ms
- 卡片悬浮：阴影加深 200ms
- 列表项点击：背景色加深 100ms

### 15.4 Loading 动画

- 旋转：360° 线性 1s 循环
- 骨架屏：渐变平移 1.5s 循环

## 16. 苗族文化元素使用规范

### 16.1 可使用的文化元素

| 元素 | 使用场景 | 使用方式 |
|------|---------|---------|
| 苗银主色 | 品牌主色 | 整页通用 |
| 蜡染靛蓝 | 装饰点缀 | 背景图案、分割线 |
| 苗绣橙 | 强调色 | 按钮、价格、徽章 |
| 苗银纹样 | Logo、装饰 | 不超过画面 20% |
| 苗年节庆插画 | Banner、启动页 | 主体展示 |
| 苗族长桌宴 | 食模块主视觉 | Banner 配图 |
| 苗寨木屋 | 住模块主视觉 | 详情页配图 |
| 梯田 | 行模块主视觉 | Banner 配图 |
| 苗族银饰纹样 | 社区/空状态插画 | 装饰元素 |

### 16.2 不可使用的元素

- ❌ 宗教、祭祀、图腾等敏感文化元素
- ❌ 对苗族文化不尊重或刻板印象的呈现
- ❌ 过度堆砌民族文化元素导致界面杂乱
- ❌ 未经授权的传统纹样复制

### 16.3 文化元素使用原则

1. **比例克制**：民族文化元素占整页不超过 20%
2. **现代化表达**：将传统纹样进行扁平化、几何化再设计
3. **情感共鸣**：唤起对苗寨的向往与好奇
4. **尊重准确**：文化元素使用前需查阅资料或咨询当地文化专家

## 17. 三端适配原则

### 17.1 微信小程序端

- 使用 **rpx** 单位（750rpx = 屏幕宽）
- 推荐尺寸按 iPhone 6（375×667）为基准
- 关键设计稿宽度：**375px**
- 顶部导航栏高度：44px（iOS）/ 48px（Android）
- 底部 Tab 高度：56px + 安全区
- 安全区适配：iPhone X 及以上需预留底部 34px

### 17.2 PC 网页端

- 最小宽度：1024px
- 推荐设计稿宽度：1440px
- 容器最大宽度：1200px，居中
- 字体最小 14px
- 鼠标悬浮态必须有视觉反馈

### 17.3 管理后台

- 最小宽度：1280px
- 推荐设计稿宽度：1440px
- 固定侧边栏 220px（可折叠到 64px）
- 顶部 Header 高度 56px
- 主内容区内边距 24px
- 表格、表单、按钮为管理后台主要交互元素

### 17.4 响应式断点

| 断点 | 宽度 | 说明 |
|------|------|------|
| xs | < 640px | 移动端（小屏手机） |
| sm | 640-768px | 大屏手机 |
| md | 768-1024px | 平板 |
| lg | 1024-1280px | PC 小屏 |
| xl | ≥ 1280px | PC 大屏 |
