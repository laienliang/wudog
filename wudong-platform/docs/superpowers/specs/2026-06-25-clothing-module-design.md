# 衣·非遗商品模块 — 设计文档

> 日期：2026-06-25
> 项目：乌东文旅"衣食住行"综合服务平台
> 状态：设计完成，待实施

---

## 1. 需求概述

为衣模块（非遗商品）提供完整的前后端功能，包括商品浏览、分类筛选、SKU 选择、评价查看、收藏、本地购物车，以及管理后台的商品/分类/订单/评价/库存管理。严格遵循《需求规格说明书》6.1-6.5 节要求。

## 2. 后端 API（增强已有 CRUD）

| 方法 | 端点 | 说明 |
|------|------|------|
| GET | `/api/v1/categories` | 商品分类列表（树形） |
| GET | `/api/v1/products` | 商品列表（分类+价格区间+风格+材质筛选+排序+分页+搜索） |
| GET | `/api/v1/products/:id` | 商品详情 |
| GET | `/api/v1/products/:id/skus` | 商品 SKU 列表 |
| GET | `/api/v1/products/:id/reviews` | 商品评价 |
| POST | `/api/v1/products` | 新增商品（商家） |
| PUT | `/api/v1/products/:id` | 更新商品（商家） |
| DELETE | `/api/v1/products/:id` | 删除商品（软删除） |
| POST | `/api/v1/products/:id/favorite` | 收藏/取消收藏 |
| GET | `/api/v1/products/search/hot` | 热门搜索词 |
| POST | `/api/v1/products/batch-import` | 批量导入商品 |

## 3. PC 端页面结构

```
src/pages/clothing/
├── index.tsx            # 商品列表页
├── detail.tsx           # 商品详情页
├── cart.tsx             # 购物车页面
├── favorites.tsx        # 收藏夹
└── my-reviews.tsx       # 评价中心
```

### 3.1 商品列表页

| 功能 | 说明 |
|------|------|
| 分类筛选 | 左侧分类树，点击筛选 |
| 多条件筛选 | 价格区间、评分、风格、材质 |
| 排序 | 按销量/价格/评分/最新排序 |
| 商品卡片 | 图片+名称+价格+评分+销量 |
| 分页 | 每页 12 个，底部分页器 |
| 搜索 | 顶部关键词搜索 + 搜索历史 + 热门搜索 |

### 3.2 商品详情页

| 功能 | 说明 |
|------|------|
| 主图轮播 | 大图展示，支持放大 |
| 商品信息 | 标题/价格/销量/评分 |
| SKU 选择 | 规格切换，价格联动 |
| 工艺介绍 | 文化故事/制作工艺 |
| 传承人信息 | 传承人姓名/介绍/头像 |
| 收藏 | 收藏/取消收藏按钮 |
| 加入购物车 | 选 SKU 后加购（localStorage） |
| 立即购买 | 直接调订单 API 下单 |
| 评价列表 | 查看用户评价（分页） |

### 3.3 购物车页面

| 功能 | 说明 |
|------|------|
| 商品列表 | 展示已加购商品 |
| 数量调整 | 加减数量 |
| 删除 | 从购物车移除 |
| 合计金额 | 自动计算总价 |
| 去结算 | 调订单 API 创建订单 |

> 购物车数据存储在 localStorage，不调用后端接口。

## 4. 管理后台页面结构

```
src/pages/clothing/
├── product-list.tsx    # 商品管理（表格+搜索+批量导入）
├── product-form.tsx    # 商品表单（新增/编辑）
├── category-list.tsx   # 分类管理
├── orders.tsx          # 订单管理
├── reviews.tsx         # 评价管理
└── inventory.tsx       # 库存管理
```

### 4.1 商品管理

| 功能 | 说明 |
|------|------|
| 商品表格 | 图片/名称/分类/价格/库存/状态 |
| 搜索筛选 | 按名称/分类/状态搜索 |
| 新增 | 弹窗表单，含图片上传 |
| 编辑 | 弹窗表单，预填数据 |
| 上下架 | 切换 status 0/1 |
| 删除 | 软删除 |
| 批量导入 | 支持 Excel/CSV 批量导入商品 |

### 4.2 分类管理

| 功能 | 说明 |
|------|------|
| 分类列表 | ID/名称/排序/操作 |
| 新增分类 | 名称+排序+图标 |
| 编辑分类 | 修改名称/排序 |
| 删除分类 | 有关联商品时禁止删除 |

### 4.3 订单管理

| 功能 | 说明 |
|------|------|
| 订单表格 | 订单号/用户/金额/状态/时间 |
| 状态筛选 | 按订单状态 Tab 切换 |
| 操作 | 发货/确认退款 |

### 4.4 评价管理

| 功能 | 说明 |
|------|------|
| 评价列表 | 商品/用户/评分/内容/时间 |
| 回复 | 商家回复评价 |
| 隐藏 | 隐藏不当评价 |

### 4.5 库存管理

| 功能 | 说明 |
|------|------|
| SKU 库存列表 | 商品/规格/SKU/当前库存 |
| 库存调整 | 手动调整库存数量 |
| 库存预警 | 库存 < 10 时红色提醒 |
